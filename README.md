# Taiphoon 技術支援中心

## 關於Taiphoon

Taiphoon 是來自台灣的無人機電子與飛控技術團隊，成立於 2022 年，並建立在超過二十年的無人機電子設計與製造經驗之上。

我們專注於開發無人機核心系統，包括飛行控制器、電子變速器（ESC）、GPS模組與包括感測器等周邊設備，提供兼具可靠性、性能與整合彈性的產品方案。

所有產品皆在台灣完成設計、製造與測試，涵蓋硬體與軟體設計、電子製造、組裝驗證到系統整合，確保每一項產品都符合實際應用所需的品質與穩定性。

我們同時長期投入開源生態，重視與 ArduPilot、PX4 等平台的相容性，協助開發者、整合商與飛手更有效率地完成部署與應用。

Taiphoon 文件中心提供 Morakot 飛控與相關產品的中文技術文件，內容涵蓋硬體接線、韌體安裝、參數設定、功能啟用與常見問題排除，協助你從首次上電到實際飛行快速完成配置。
