# 安裝馬達(Motors)



{% hint style="info" %}
在安裝馬達前應該注意機架上面的鎖孔是否與馬達相符
{% endhint %}

安裝前需要在螺絲上面塗上螺絲膠後再鎖進去，讓馬達更加固定

安裝完成後，搖動馬達，確認沒有晃動與異音後即可進行電子調速器ESC安裝
